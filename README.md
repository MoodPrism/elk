# ELK Stack for MoodPrism Project

## How to start:

#### RUN PROJECT FIRST (elk's root directory)
See MoodPrism/project for details

#### RUN DOCKER-COMPOSE 
```
docker-compose up
```
## How to close:
```
docker-compose down
```
